介绍
----------

三国杀是现行比较流行的棋牌游戏，也是网络游戏，而本项目尝试克隆网络版三国杀。整个项目用C++编写，使用Qt图形框架作为游戏引擎。我们曾经尝试过其他开源游戏引擎，如SDL，HGE，Clanlib等，但这些引擎缺乏一些重要功能。虽然Qt是一个应用程序框架而不是游戏引擎，但是它的图形框架适合开发这个游戏。

特征
----------

1.框架
* 使用 Qt 图形框架开源
* 使用 FMOD 作为声音引擎
* 使用 plib 作为后端操控
* 使用 Lua 作为AI脚本和扩展编写

2.操作经验
* 完整包（包括所有官方扩展包）
* 可以将牌拖动到目标来使用牌
* 键盘增加快捷方式
* 卡牌分类（按卡牌类型和卡牌花色）
* 卡牌超过上限时的多层显示

3.扩展
* 会基于此游戏提供的一些 MOD
