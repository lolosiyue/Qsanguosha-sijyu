# 手牌篩選功能說明

## 概述
本功能提供快速選擇手牌的能力，透過分類層級幫助玩家在眾多手牌中迅速定位並選擇特定卡牌。

### 主要特性
- **兩層分類**：第一層按卡牌類別顯示代表卡，第二層顯示該類別下的所有具體卡牌
- **即時翻譯**：卡牌名稱使用 `Sanguosha->translate()` 顯示本地化文字
- **浮動標籤**：代表卡下方顯示卡牌類別名稱與數量
- **返回/關閉按鈕**：每層皆提供導航操作

## 程式碼結構

### 1. dashboard.h 修改

**新增前向宣告**
```cpp
class CardContainer;
class Button;
```

**新增 Slots**
```cpp
void showCardFilterContainer();    // 顯示篩選容器（第一層）
void filterCardChosen(int card_id); // 處理卡牌選擇
void hideFilterContainer();         // 隱藏容器
```

**新增成員變數**
```cpp
CardContainer *_m_filterContainer;           // 篩選容器
QString _m_filterCurrentCategory;             // 當前類別（空=第一層）
QList<QGraphicsItem *> _m_filterUIElements;   // UI 元素列表（標籤、按鈕）
void clearFilterUIElements();                 // 清理 UI 元素
```

### 2. dashboard.cpp 修改

**新增 includes**
```cpp
#include "cardcontainer.h"
#include "button.h"
#include <QGraphicsTextItem>
```

**建構子初始化**
```cpp
_m_filterContainer = nullptr;
_m_filterCurrentCategory = "";
```

**_createExtraButtons() 建立按鈕**
```cpp
m_btnFilterCard = new QSanButton("handcard", "filter", this);
m_btnFilterCard->setPos(pos, height);
m_btnFilterCard->setZValue(100);
connect(m_btnFilterCard, SIGNAL(clicked()), this, SLOT(showCardFilterContainer()));
```

**showCardFilterContainer() - 第一層**
- 分類統計可用手牌
- 僅取每類第一張作為代表顯示
- 容器上方顯示浮動標籤（類別名稱 + 數量）
- 右下角顯示「關閉」按鈕

**filterCardChosen() - 第二層邏輯**
- `_m_filterCurrentCategory` 為空：處於第一層
  - 單張卡牌：直接選中並關閉
  - 多張卡牌：進入第二層，顯示該類所有卡牌
- 非空：處於第二層
  - 點擊卡牌：選中並關閉
  - 提供「返回」與「關閉」按鈕

**clearFilterUIElements() - 記憶體管理**
```cpp
void Dashboard::clearFilterUIElements()
{
    qDeleteAll(_m_filterUIElements);  // 釋放記憶體
    _m_filterUIElements.clear();
}
```

### 3. cardcontainer.h 修改

**新增 getter**
```cpp
QList<CardItem *> getItems() const { return items; }
```

## 功能流程

| 操作 | 結果 |
|------|------|
| 點擊篩選按鈕 | 顯示第一層：各類別代表卡 + 浮動標籤 |
| 點擊代表卡（僅1張） | 直接選中手牌，容器關閉，卡牌升起 |
| 點擊代表卡（多張） | 進入第二層，顯示該類所有卡牌 |
| 點擊返回按鈕 | 回到第一層 |
| 點擊關閉按鈕 | 直接關閉容器 |
| 點擊第二層卡牌 | 選中手牌，容器關閉，卡牌升起 |

## 信號連接策略

使用 `Qt::QueuedConnection` 確保非同步執行，避免在滑鼠事件處理中直接刪除物件導致崩潰：

```cpp
// 容器選擇信號
connect(_m_filterContainer, SIGNAL(item_chosen(int)), this, SLOT(filterCardChosen(int)), Qt::QueuedConnection);

// 按鈕點擊信號
connect(backBtn, SIGNAL(clicked()), this, SLOT(showCardFilterContainer()), Qt::QueuedConnection);
connect(closeBtn, SIGNAL(clicked()), this, SLOT(hideFilterContainer()), Qt::QueuedConnection);
```

---

## UI 元素銷毀時機

| 時機 | 處理 |
|------|------|
| 選擇卡牌後 | 先斷開卡牌信號、禁用滑鼠響應，再清除 UI 元素，最後清除容器 |
| 返回上一層 | `clearFilterUIElements()` 後重新建立 UI |
| 關閉容器 | `clearFilterUIElements()` + `_m_filterContainer->clear()` + `_m_filterContainer->hide()` |
